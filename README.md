# HTTPExp – Single-File PHP Web File Manager + Terminal

Ultra-lightweight (~350 lines) all-in-one PHP webshell / backdoor file manager with built-in terminal.  
Zero dependencies · No external CDNs · Works on PHP 5.3 → 8.x

### Features
- Browse, upload (drag & drop), edit, rename, delete files/folders  
- Full terminal (execute any system command)  
- Simple password protection  
- Path traversal protection  
- Retro green hacker style (fully self-contained)

### Usage
1. Edit line 5 → change the password (`$PASSWORD = "admin123";`)
2. Upload as `httpexp.php` on target
3. Access → enter password → profit
